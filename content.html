In Python the print function is used to display output. This can be useful for reporting results to the user or debugging code. To use it, write the word print followed by parentheses containing the values you wish to print, such as in the example below:

print(42)

You can also provide multiple values to the print function by separating them with commas. The values will be printed with a space in between them by default:

print(999, 42)

By default, the print function adds a newline character at the end of the output, which means that subsequent print statements will appear on a new line:

print(10)
print(9)
